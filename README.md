# 빠른 실행

1. 프로젝트 클론 받기
```
git clone {프로젝트 url}
```

2. 의존성 설치
```
npm install
npm install axios
npm install vue-router
npm install xlsx
npm install vue3-excel-editor  //엑셀 ui 라이브러리
npm install ag-grid-community ag-grid-vue3  
npm install pinia  // pinia 상태관리
npm install pinia-plugin-persistedstate  // pinia 상태 지속 플러그인

```

3. 프로젝트 실행
```
npm run dev
```
